**Wilson Wambua | Cybersecurity Portfolio**

A personal cybersecurity portfolio website built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies beyond Font Awesome and Google Fonts.

**Overview**

This portfolio showcases my journey as a Bachelor of Science student in Information Security and Forensics at KCA University, with a focus on SOC operations, ethical hacking, digital forensics, and network security. The site is designed around a dark hacker terminal theme — monospace fonts, green/cyan glow effects, and a subtle scanline overlay to reflect the cybersecurity aesthetic.

**Sections**

- **About** — Professional bio and quick-stats sidebar
- **Resume** — Tabbed view of education, experience, skills, and certifications
- **Projects** — Six security projects including LIDFT (log-based intrusion detection), a browser hijacker scanner, and an NIDS dashboard
- **Skills & Tools** — Categorised tags covering 15+ security tools, languages, and concepts
- **Certifications** — Verified badge cards for Cisco, CyberShujaa, and TryHackMe credentials
- **Labs / CTF** — Documented lab challenges from TryHackMe, Hack The Box, and Blue Labs
- **Contact** — Terminal-style form and direct links to all social/professional profiles

**Tech Stack**

- HTML5, CSS3, Vanilla JavaScript
- Google Fonts — JetBrains Mono, Fira Code
- Font Awesome 6 icons
- No build tools or frameworks required — drop `index.html` in a browser and it works

**Features**

- Fully responsive (mobile + desktop)
- Typing animation on hero name
- Tab-switching resume system
- Smooth scroll with active nav tracking
- CSS glow effects and scanline overlay
- Zero external JavaScript dependencies
